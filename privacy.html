🔒 Kebijakan Privasi

1. Informasi yang Kami Kumpulkan

✅ NotaKu TIDAK menyimpan data pribadi Anda di server manapun.

Data hanya digunakan untuk menampilkan nota, statistik penjualan, laporan laba/rugi, dan manajemen stok di perangkat Anda sendiri. Tidak ada pihak ketiga yang dapat mengakses data tersebut.

3. Keamanan Data

✅ Website dilindungi dengan SSL/HTTPS (koneksi aman terenkripsi).

✅ Kami TIDAK akan pernah meminta password, PIN, atau data sensitif lainnya.

✅ Data hanya bisa diakses oleh Anda (tidak ada pihak ketiga).

✅ Transfer pembayaran hanya melalui rekening BRI 3593-0100-9186-508 atas nama Ardian Nurraihan.

✅ Kami tidak menyimpan data kartu kredit atau informasi pembayaran lainnya.

✅ Bukti transfer yang dikirim via WhatsApp hanya untuk verifikasi dan akan dihapus setelah kode dikirim.

5. Cookie dan Tracking

✅ Website menggunakan Google Analytics untuk statistik pengunjung (anonim).

✅ Tidak ada cookie pelacakan dari pihak ketiga lainnya.

6. Backup & Restore Data

✅ NotaKu menyediakan fitur backup data ke file JSON.

✅ File backup disimpan di perangkat Anda, bukan di server kami.
